# РГР за 2ой семестр по дисциплине "Программирование"

## Описание проекта
Этот проект представляет собой расчетно-графическую работу по программированию на C++, посвященную реализации криптографических алгоритмов для шифрования и дешифрования файлов.
В программе использованы три шифра: 
- Атбаш (отражение байтов 0..255)
- Аффинный (a*x + b mod 256)
- XOR (побитовое XOR с повторяющимся ключом)

Приложение позволяет загружать файлы, генерировать или вводить ключи, выполнять шифрование/дешифрование и сохранять результаты. Разработано для демонстрации принципов криптографии в образовательных целях.

## Цели
1. Изучить и реализовать классические криптоалгоритмы для работы с бинарными данными.
2. Обеспечить удобный консольный интерфейс с обработкой ошибок.
3. Продемонстрировать эволюцию методов шифрования от исторических до современных.

## Функции
- Выбор режима (шифрование/дешифрование).
- Выбор шифра (Атбаш, Аффинный, XOR).
- Загрузка файла для обработки.
- Ввод, чтение из файла или генерация ключа (с проверками валидности).
- Выполнение шифрования/дешифрования с индикатором прогресса.
- Сохранение результата и ключа в файл (с опцией перезаписи).
